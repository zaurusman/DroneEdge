Session naming

Delete recordings

Long-press a gallery row to delete a recording (video + JSON). Confirmation dialog before delete.

Video thumbnails in gallery

Show a small frame grab next to each recording row so you can visually identify sessions at a glance.

C

Live recording timer

While recording is armed, show a running elapsed time (e.g. "● REC 0:42") on the live screen HUD.

D

Detection count in gallery

Show how many detection events were logged during each session (read from the JSON sidecar).

E

Share / export recording

Share button per gallery row to send the MP4 to another app (Files, Drive, etc.) via Android share sheet.
